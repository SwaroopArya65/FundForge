<h2>📌 Overview</h2>

<p>
FundForge is a crowdfunding-style platform built for student innovators to showcase startup ideas and receive support from the community.
It simulates real-world fundraising to help validate ideas and boost confidence before real launch.
</p>

<hr/>

<h2>🎯 Features</h2>

<h3>👨‍🎓 Student</h3>
<ul>
  <li>Create and manage campaigns</li>
  <li>Set funding goals & deadlines</li>
  <li>Track real-time progress</li>
</ul>

<h3>💰 Supporter</h3>
<ul>
  <li>Explore campaigns</li>
  <li>Pledge support</li>
  <li>Track contributions</li>
</ul>

<h3>🛠️ Admin</h3>
<ul>
  <li>Approve/reject campaigns</li>
  <li>Monitor platform activity</li>
  <li>Manage users</li>
</ul>

<hr/>

<h2>🔐 Authentication & Security</h2>

<ul>

<h2>⚙️ Installation</h2>

<pre>
git clone https://github.com/your-username/fundforge.git
cd fundforge
</pre>

<h3>Backend</h3>

<pre>
cd server
npm install
npm run dev
</pre>

<h3>Frontend</h3>

<pre>
cd client
npm install
npm run dev
</pre>
